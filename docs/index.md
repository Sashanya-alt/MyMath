# Реализация класса матрицы
Данный проект реализует класс матрицы на языке C# в рамках лабораторной работы по дисциплине "Технология разработки программного обеспечения"
## Функционал
Данный проект реализует следющие действия с матрицами:

- Создание объектов класса матрицы заданных размеров посредством инициализации через константное значение или через multidimensional array
- Простейшие операции с матрицами, такие как сложение, отрицание, вычитание, умножение на число или перемножение матриц
